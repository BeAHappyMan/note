+ 下载ns3

  ```shell
  mkdir ns3 
  cd ns3
  git clone https://gitlab.com/nsnam/ns-3-allinone.git
  cd ns-3-allinone 
  python3 download.py -n ns-3.40 
  ```

+ 构建

  ./ns3 configure --enable-examples --enable-tests

  ./ns3 build
